Node.js

Node.js is an open source, cross-platform runtime environment command line tool. It was built for server side JavaScript code.

One of the things that really sticks out in Node.js is that it is able to operate on a single-thread. Traditional server side techniques requires each request to spawn a new thread; this causes the app to take up system RAM, and eventually it maxes out at the number of RAM available.

Node.js uses non-blocking I/O (input/output) calls and is able to support tens of thousands of concurrent connections. Every I/O call that Node.js makes must immediately take a callback which keeps the CPU busy, and can save you memory. If the call requires time to complete, then Node.js will continue running until the completion of the I/O is later communicated back. The non-blocking I/O call waits for no one!
